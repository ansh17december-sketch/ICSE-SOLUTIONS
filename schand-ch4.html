7. (iv), (v), (vi): Solution by Cases (sample)
(iv) x + y = 2 ; {0,1,2,3}
If x = 0, then 0 + y = 2 ⇒ y = 2
If x = 1, then 1 + y = 2 ⇒ y = 1
If x = 2, then 2 + y = 2 ⇒ y = 0
If x = 3, then 3 + y = 2 ⇒ y = –1
So solution set: {(0,2), (1,1), (2,0)}
------- (v) 3x + 2 < 4 ; x = –4,–3,–2,–1,0,1,2,3,4
Try each x:
x = –4, 3(–4)+2=–10 < 4 ✔
x = –3, 3(–3)+2=–7 < 4 ✔
x = –2, 3(–2)+2=–4 < 4 ✔
others fail. Solution set: {–4,–3,–2}
------- (vi) 2(x–3) < 1 ; {2,3,4,10}
x=2: 2(2–3)=–2 < 1 ✔
x=3: 2(0)=0 < 1 ✔
x=4: 2(1)=2 < 1 ✗
x=10: 2×7=14<1 ✗
Solution set: {2,3}
8. Triangle Sides (Parts i & ii)
(i) For triangle sides x, x+1, x+2:
x + (x+1) > x+2 → x > 1
x+1 + x+2 > x → x+3 > 0 ⇒ x > –3
x+(x+2) > x+1 → x+2 > x+1 ⇒ 1>0 always
(ii) So for integers: x > 1 ⇒ x = 2,3,4...

So –3 ≤ x ≤ 8/3.
