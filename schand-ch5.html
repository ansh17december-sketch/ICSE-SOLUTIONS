Question 18.
a/(a + b) + b/(a - b) = (a - b)/(a + b) Multiply by (a + b)(a - b): a(a - b) + b(a + b) = (a - b)² a² - ab + ab + b² = a² - 2ab + b² a² + b² = a² - 2ab + b² 0 = -2ab ab = 0 Either a = 0 or b = 0
Question 19.

Question 20.
